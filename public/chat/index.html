Browser Client

Use this MCP server without an IDE.

This chat app calls the local MCP endpoint through a streaming MCP client and shows live progress while the Java expert responds.

Current flow

  1. Paste a GitHub token.
  2. Ask a Java question.
  3. Watch MCP progress and streamed answer text.

Connection

The token stays in your browser only. It is sent to the local server so the MCP client can authenticate against /mcp.
